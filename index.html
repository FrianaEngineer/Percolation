Overview

What Percolation Means

Percolation is a way to study how something moves through a system. In this project, the system is shown as a grid made of small squares. Each square can either be blocked or open. If enough open squares connect from the top of the grid to the bottom, then the system percolates. This means there is a complete path that allows something, like water or electricity, to travel through.

Example

Water and Electricity

A simple way to imagine percolation is water moving through soil. If there are enough connected empty spaces in the soil, water can pass through from the surface to deeper layers. If the spaces are blocked or disconnected, the water cannot flow all the way through. The same idea can also apply to electricity. If enough conductive materials are connected, electricity can travel through the object. If not, the material stays insulated.

Threshold

When a Path Appears

Percolation is important because it helps us understand when a connected path forms in a random system. At first, only a few spaces might be open, so nothing can move very far. But as more spaces open, the system eventually reaches a point where a full path suddenly appears. This point is called the percolation threshold.

Applications

Why It Matters

Percolation can model water draining through land, oil moving through rock, diseases spreading through populations, electricity passing through materials, or information spreading through networks. In computer science, it turns a flow problem into a connectivity problem.

Monte Carlo Simulation

Estimating the Percolation Threshold

Start with every site blocked. Then repeatedly choose one blocked site uniformly at random and open it. When the system first percolates, record the fraction of sites that are open. That fraction is one estimate of the percolation threshold.

By repeating this experiment many times and averaging the results, we get a more reliable estimate. The simulation below reports the sample mean, sample standard deviation, and a 95% confidence interval.

Mean: μ = (x₁ + x₂ + ... + xT) / T

Std. dev.: σ = sqrt( Σ(xᵢ - μ)² / (T - 1) )

95% CI: μ ± 1.96σ / sqrt(T)
