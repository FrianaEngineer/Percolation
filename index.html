Percolation Explorer

A compact website for presenting the percolation model and interacting with the same open-site visual logic used in the Java visualizer.

Overview

What Percolation Means

Percolation is a way to study how something moves through a system. In this project, the system is shown as a grid made of small squares. Each square can either be blocked or open. If enough open squares connect from the top of the grid to the bottom, then the system percolates. This means there is a complete path that allows something, like water or electricity, to travel through.

Example

Water and Electricity

A simple way to imagine percolation is water moving through soil. If there are enough connected empty spaces in the soil, water can pass through from the surface to deeper layers. If the spaces are blocked or disconnected, the water cannot flow all the way through. The same idea can also apply to electricity. If enough conductive materials are connected, electricity can travel through the object. If not, the material stays insulated.

Threshold

When a Path Appears

Percolation is important because it helps us understand when a connected path forms in a random system. At first, only a few spaces might be open, so nothing can move very far. But as more spaces open, the system eventually reaches a point where a full path suddenly appears. This point is called the percolation threshold. It tells us about the moment when the system changes from not connected to connected.

Applications

Why It Matters

This connects to real-world problems because many systems depend on whether paths exist. Scientists can use percolation to model water draining through land, oil moving through rock, diseases spreading through populations, electricity passing through materials, or even information spreading through networks. In computer science, percolation is useful because it turns a real-world movement problem into a connectivity problem. By checking which parts are connected, we can predict whether the system allows flow from one side to another.
